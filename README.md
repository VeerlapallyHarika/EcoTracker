EcoTracker – Carbon Footprint & Green Habits

EcoTracker is a simple, user-friendly web application designed to help users track their carbon footprint, adopt eco-friendly habits, and build sustainable awareness.
The project focuses on clean UI, beginner-friendly logic, and practical environmental impact.

🚀 FEATURES:

 Carbon Footprint Calculator
Estimates carbon emissions based on daily activities

 Green Habits Tracker
Encourages eco-friendly actions with progress tracking

 Visual Feedback
Displays results and habit progress clearly

Beginner-Friendly Design
Easy to understand, clean layout, and responsive UI

Educational Focus
Raises awareness about sustainability and climate responsibility

🚀Technologies Used:
HTML5 – Structure of the application

CSS3 – Styling, layout, and theme variables
JavaScript (Vanilla JS) – Logic, calculations, and interactivity

🚀Use Case

Students learning HTML, CSS, and JavaScript

Mini-projects for college submissions

Beginners exploring sustainability-based applications

Hackathon or portfolio projects
